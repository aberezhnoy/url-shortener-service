URL Shortener REST Service

Simple in-jvm storage URL shortener REST service. All data stored in memory.
post /account

Create new account

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body (required)
    {
        "accountId": String (required)
    }

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200 - OK

409 - Duplicate account

412 - Precondition fail

post /register

Register a new URL

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body (required)
    {
        "url": String (required)
        "redirectType": Number 301|302, default 302 (optional)
    }

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200 - OK

401 - Unauthenticated

409 - Duplicate URL

412 - Precondition fail

get /statistic/{accountId}

Retrieves URL redirection statistics for account

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200 - OK

401 - Unauthenticated

403 - Forbidden

get /{shortUrlId}

Perform redirect to URL and update statistics

Responses

301 - Moved Permanently

302 - Found
